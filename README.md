# 🚀 Romulo's Developer Portfolio

Bem-vindo ao repositório do meu portfólio pessoal online! Este site foi desenvolvido com o propósito de apresentar meus projetos, habilidades técnicas e oferecer um canal direto de contato profissional.

## 🛠️ Tecnologias Utilizadas

Este portfólio foi construído com foco em leveza e performance, sem necessidade de back-end:

* **HTML5:** Estruturação semântica de todo o conteúdo.
* **CSS3 Customizado:** Design premium focado em efeitos modernos, especificamente *Glassmorphism* (efeito de vidro translúcido).
* **Bootstrap 5:** Framework para garantir responsividade perfeita em celulares, tablets e desktops de forma fluida.
* **JavaScript (Vanilla):** Scripts leves para micro-animações, como o efeito de `fade-up` ao rolar a página.
* **Formspree:** API externa que cuida de toda a lógica do formulário de contato, enviando mensagens diretas para meu E-mail sem precisar de infraestrutura complexa.

## ✨ Funcionalidades

- **Design Premium:** Coletânea de recursos de front-end com um Dark Theme moderno, gradientes, e micro-animações.
- **Responsivo (Mobile-First):** Layout desenhado para adaptar-se confortavelmente a qualquer formato de tela.
- **Visualização de Projetos:** Uma vitrine de trabalhos (especialmente jogos estilo Tower Defense renderizados na Unity Engine).
- **Formulário Funcional:** Um formulário de contato que me envia uma notificação no exato momento que um cliente enviar uma mensagem.

## 🌐 Como visualizar meu projeto

Por ser um site 100% estático, visualizar este projeto não exige instalações trabalhosas de pacotes como Node.js ou compiladores Python:

1. Faça o clone deste repositório usando `git clone https://github.com/Romulo-ns/my_portfolio.git`.
2. Acesse a pasta do portfólio.
3. Clique duas vezes no arquivo `index.html`. O arquivo será aberto diretamente pelo navegador (Edge/Chrome/Firefox).

## 🛜 Hospedagem (Opcional - GitHub Pages)

O próximo passo é ativar via **GitHub Pages** totalmente de graça:
1. Vá até as "Settings" deste repositório no GitHub.
2. Na barra lateral à esquerda, clique em "Pages".
3. Em "Source" (Source/Branch), selecione `main` e "Save".
4. O GitHub hospedará o arquivo ao vivo!
